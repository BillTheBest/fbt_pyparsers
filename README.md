# Python FBT parsers

## Requirements
- python >= 3.4
- python packages:
  - `requests`

## Run the parser

```bash
python parser_runner.py
```

## Options

When running `parser_runner.py` from command line, you can use the following arguments:

- `-h`, `--help`: View help
- `-d`, `--debug`: Activate debug mode
- `-v`, `--verbose`: Activate verbose mode
